
# Portfolio Web – Imane Mekkaoui

## Présentation du projet

Ce projet consiste à développer un **portfolio web personnel** en utilisant uniquement **HTML5 et CSS3**.
L’objectif est de mettre en pratique les notions fondamentales du développement web front-end, notamment la structure sémantique d’une page HTML, la mise en page avec CSS, ainsi que la création d’un design simple et responsive.

Le site présente des informations personnelles, les compétences techniques, quelques projets réalisés ainsi que des moyens de contact.

---

## Objectifs du projet

* Concevoir un site web personnel structuré.
* Appliquer les bonnes pratiques HTML5.
* Utiliser CSS pour améliorer l’apparence du site.
* Mettre en place une mise en page avec **Flexbox ou Grid**.
* Créer un site **responsive** qui s’adapte aux écrans mobiles.

---

## Technologies utilisées

Les technologies suivantes ont été utilisées pour réaliser ce projet :

* **HTML5** : structure et organisation du contenu
* **CSS3** : mise en forme et design du site
* **Flexbox / Grid** : gestion de la mise en page
* **Media Query** : adaptation du site aux différents écrans
* **Git & GitHub** : gestion de version et partage du projet

---

## Structure du projet

Le projet est organisé de la manière suivante :

```
portfolio/
│
├── index.html
├── style.css
├── images/
│   └── photo.jpg
└── README.md
```

* **index.html** : fichier principal contenant la structure du site
* **style.css** : fichier contenant les styles CSS
* **images/** : dossier contenant les images utilisées dans le portfolio
* **README.md** : description du projet
